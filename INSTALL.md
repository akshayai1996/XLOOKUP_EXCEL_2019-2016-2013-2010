# XLOOKUP Installation Guide

## 1. Check your Excel Version
1. Open Excel -> **File** -> **Account** -> **About Excel**.
2. Look at the top line for **32-bit** or **64-bit**.

## 2. Locate the File
- For 64-bit Excel: Use `XLookup_64bit.xll`
- For 32-bit Excel: Use `XLookup_32bit.xll`

## 3. Install
1. Open Excel -> **File** -> **Options** -> **Add-ins**.
2. Manage: **Excel Add-ins** -> **Go...**.
3. **Browse...** -> Select the `.xll` file -> **OK**.
4. Click **Yes** to copy to library.
5. Ensure it is checked and click **OK**.

## 4. Verify
Type `=XLOOKUP(` in any cell.
